Portfolio fully redesigned for SecOps/Cybersecurity profile with "Threat Operations Interface" aesthetic.

**Why:** User requested creative redesign aligned with cybersecurity/cloud profile. New aesthetic: dark terminal, monospace font (JetBrains Mono), grid background, color-coded status badges (red=offensive, green=active, purple=SecOps).

**New page structure:**
1. Home — terminal hero with ONLINE status, stats row, Arsenal logowall
2. Operations — terminal-window cards for security projects (OP-001..OP-006)
3. Areas — 6 expertise cards (SecOps, VPM, OSINT/DarkINT, Pentesting, DevSecOps, SecArch)
4. Experience — timeline with pulse dots for active roles
5. Community — OpenVector Intelligence initiative + GDG Speaker + PwnedCR + CNCF KCD + GDG
6. Certificates — IRSI, ISC2, Linux Foundation, SCRUM
7. Footer/Contact — integrated contact form + focus areas

**Key new content:**
- OffsecAssistant AI Agent (OP-001) — AI for offensive security operations
- OpenVector Intelligence — DarkINT/OSINT threat hunting initiative
- OSINT & DarkINT area (includes Tor network threat investigations)
- PwnedCR volunteer
- GDG Speaker (2026)
- GDG Volunteer

**CSS design system:** --red (#ff3a3a), --green (#00ff87), --sec (#a476ff), --terminal-bg, JetBrains Mono via .mono class, grid background via .grid-bg pseudo-element, tag-red/tag-green/tag-purple/tag-gray utility classes.
